Requirements:
    1. Python version 3.12 (min)
    2. Knowledge on LLM, A2A Protocol



Open 2 separate terminals & follow the below instructions.
->  For Client side:
        1. cd a2a_samples
        2. cd client
        3. python time_client.py

->  For Server side:
        1. cd a2a_samples
        2. cd server
        3. python tell_time_server.py
